# hackatonPAWeb

MusicBook es una red social de músicos, dondé se pueden encontrar diferentes artistas, conocedores de música ó simplemente
aficionados de la música y sus diferentes apreciaciones.

Se utiliza el framework CodeIgniter version 3.1.6 para facilitar las consultas a bases de datos, tener un manejo sencillo
de las vistas, modelos y controladores de la página, también, con la implementacion de este framework se busca crear una
aplicacion estable.

Si quieres encontrar amigos, compañeros para una banda, o simplemente personas que gusten del mismo tipo de musica o 
instrumentos, MusicBook es para ti.

Requerimientos
-Navegador de Internet
	-Se recomienda Opera, Chrome, Mozilla FireFox
-Xampp

Aclaraciones pertinentes:
	MusicBook no esta pensado para ser liberado como una app, esta aplicacion esta pensada con fines academicos 
y no se espera lucrar con ella.
